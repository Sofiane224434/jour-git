Catégorie Commande À quoi ça sert
BashpwdAfficher le chemin du dossier courant.
BashlsLister les fichiers du dossier.
Bashcd mon-dossierEntrer dans un dossier.
Bashcd ..Revenir au dossier parent.
Bashcd "C:\Users\Hp\Desktop\git commit"Aller dans un chemin avec espaces (guillemets requis).
Bashclear / clsNettoyer l’écran du terminal (bash / PowerShell).
Gitgit initInitialiser un dépôt Git dans le dossier courant.
Gitgit add .Ajouter tous les fichiers à l’index (préparation du commit).
Gitgit commit -m "message"Créer un commit local avec un message.
Gitgit statusVoir l’état des fichiers (modifiés, en attente, etc.).
Gitgit log --onelineConsulter l’historique sous forme compacte.
Gitgit remote add origin <url>Lier le dépôt local à GitHub (remote origin).
Gitgit branchAfficher la branche courante.
Gitgit branch -M mainRenommer la branche actuelle en main.
Gitgit push -u origin mainEnvoyer main sur GitHub et créer le suivi (premier push).
Gitgit pushEnvoyer les nouveaux commits vers le dépôt distant.
Gitgit pullRécupérer les changements distants.
Fixgit pull origin main --rebaseIntégrer d’abord les changements distants si le push est rejeté (fetch first).
Fixgit add . + git commit -m ...Créer un premier commit si « src refspec main does not match any ».
